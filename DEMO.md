# Demo Guide

## Preview
```bash
# Install Shopify CLI: https://shopify.dev/docs/themes/tools/cli
shopify theme dev --store=your-store.myshopify.com
```

## What You See
- All sections render with default settings from schema blocks
- Theme editor sidebar shows all configurable options
- Mobile responsive layout works out of the box

## Customization
- Open the Shopify theme editor in your store admin
- Each section has configurable settings (colors, text, images)
- JSON templates control section ordering
